# Dragneel-TextEditor

## A text editor written by a Novice in C (WIP)

### Requirements
- **gcc**
- **make** 

### Installation
- **Clone the repository**
```
git clone https://github.com/BlackPhoenix/Portfolio
cd Portfolio/Dragneel-TextEditor/
```

- **Create the binary**
```
make
sudo cp Dragneel /usr/local/bin/
```
